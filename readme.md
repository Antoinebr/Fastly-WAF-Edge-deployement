# Fastly WAF @Edge deployement

## Install 

```
npm install
```

## Setup 

Copy the ```.env.sample``` and rename it ```.env``` then fill the informations with your own credentials. 

```
cp .env.sample  .env
```

## Run 

```
npm run cli
```
